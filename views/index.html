API Timestamp microservice

Make a GET Request with a timestamp or natural language to get in return the correspondance in JSON format.

 See the code on Github

User stories:
  1. With a GET Request, you can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date (example: January 1, 2016)
  2. If it does, it returns both the Unix timestamp and the natural language form of that date.
  3. If it does not contain a date or Unix timestamp, it returns null for those properties.

Example usage:

https://timestamp-idnove.herokuapp.com/api/April%2015,%202016
Test it

https://timestamp-idnove.herokuapp.com/api/1460678400
Test it
